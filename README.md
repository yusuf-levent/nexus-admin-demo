# Nexus Admin Demo

Premium SaaS admin dashboard demo built for portfolio and gig-cover use.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts
- Cloudflare Pages static export

## Screens

- Login
- Dashboard with KPI cards, line chart, bar chart, pie chart, and transactions
- Users with search, filters, avatars, status tags, and pagination
- Products and orders table
- Settings form with toggles
